4. arrow function(화살표함수)
- 함수 표현식보다 단순하고 간결한 문법으로 함수를 만들 수 있으며 ES6버전 이상 사용가능
- this나 super에 대한 바인딩이 없고, methods 로 사용될 수 없습니다.
- new.target키워드가 없습니다.
- 일반적으로 스코프를 지정할 때 사용하는 call, apply, bind methods를 이용할 수 없습니다.
- 생성자(Constructor)로 사용할 수 없습니다.
- yield를 화살표 함수 내부에서 사용할 수 없습니다.

let arrowFunc = 매개변수 or 괄호 사용(생략가능) =>{
console.log("화살표표기함수")
}

5. 즉시실행함수
- 정의되자마자 즉시 실행되는 함수. 괄호( () )로 함수를 감싸서 실행하는 문법이다.
(function (){
i=0;
while(i<10){
document.write(i);
i+=1;
}
})();

(function(){
console.log("바로실행!")
} ())

6. 재귀함수
- 함수가 자기 자신을 호출하는 함수.
재귀함수는 종료조건이 있어야 하며, 종료조건을 설정해주지 않으면 무한 반복을 하게된다.

함수가 자신을 참조하는 방법
1. 함수의 이름
2. arguments.callee
3. 함수를 참조하는 범위 내 변수

function loop(x) {
if (x >= 10) // "x >= 10" 는 탈출 조건 ("!(x < 10)"와 동등)
return;
// do stuff
loop(x + 1); // the recursive call
}
loop(0);

7. 중첩함수
- 특정 함수의 내부에 선언된 함수(함수 내부에서만 지역변수처럼 사용)
function sayHiBye(firstName, lastName) {
function getFullName() {
return firstName + " " + lastName;
}

alert( "Hello, " + getFullName() );
alert( "Bye, " + getFullName() );
}

※ 중첩함수와 클로저
- 중첩함수의 내부 함수는 클로저를 형성할 수 있으며
외부 함수는 내부 함수의 인수와 변수를 사용할 수 없는 반면에, 내부 함수는 외부 함수의 인수와 변수를 사용할 수 있다.
